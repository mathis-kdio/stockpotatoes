Téléchargement - Installation

 

 

Installation en 6 étapes

Conseils aux débutatnts
0

Il vous faut un serveur type Apache Php/Mysql

Attention, l'upload de fichiers nécessite une version de Php récente. (minimum PHP 4.3)

 

Vous pouvez par exemple installer le Kit EasyPhp qui vous fournira le nécessaire (Serveur Apache / MySql )

(télécharger ici) ou voir sur http://www.easyphp.org/

Réalisez une installation par défaut.

 

 
1

Télécharger l'archive puis décompresser sur votre serveur local ou distant.

Pour une mise à jour, recopier les fichiers de l'archive sauf le dossier Exercices. Il vous faut en effet conserver votre propre dossier Exercices qui contient les exercices déjà en ligne.

 

Je vous invite à placer le dossier stockpotatoes à la racine de votre site.

Si vous venez d'installer EasyPhp, vous copiez ce dossier Stockpotatoes (et pas seulement le contenu) à la racine de votre site c'est à dire dans le dossier portant le nom "www" de EasyPhp.

 
2

 

Aller à l'adresse http://nom_serveur/stockpotatoes/install/

 

En intranet, après installation de EasyPhp, vous pouvez laisser les paramètres indiqués dans l'étape 1 (utilisateur root et pas de mot de passe)

 

 

DROITS EN ECRITURE

L'installation tentera de donner les droits en écriture sur les 3 fichiers suivants :

  • Le dossier Exercices (voir ci-dessous)
  • Le fichier administrateur/gestion_pass.inc.php (mots de passe)
  • Le fichier Connections/ conn_intranet.php (paramètres de connection)

Cette opération n'est pas toujours possible pour certains hébergeurs. A vérifier donc... (chmod 777)

Vos exercices seront rangés ultérieurement et automatiquement dans le dossier Exercices selon l'arborescence ci-dessous

 

 

Il est impératif de posséder les droits en écriture sur ce dossier Exercices.

SI VOTRE HEBERGEUR EST FREE

Vous n'avez pas a vous soucier des problèmes de droits.

Vous devez bien évidemment avoir activé votre base MySql pour cet hébergeur avant toute installation. (Voir ici)

Vous devrez impérativement créer un dossier vide nommé sessions et cela à la racine de votre site Free (et non pas dans votre dossier stockpotatoes).

 

 

 
5

 

Intégrer le fichier des élèves - Bien relire ce document

 
 
6

Modifier éventuellement les mots de passe enseignant, administrateur...Voir Espace Administrateur

Une fois l'installation terminée, sécurisez le dossier install soit en le renommant ou en modifiant le fichier Install/.htaccess
